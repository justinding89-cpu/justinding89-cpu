## 📌 项目方向 | Project Areas

### 🧠 Generative AI & Deep Learning

- 基于深度学习的古诗词自动生成系统  
- 利用神经网络模型学习古诗词语义、韵律与风格特征  
- 实现面向文本内容生成的智能创作与语言建模应用  

### 🎬 Diffusion-based Video Generation

- 基于扩散模型的视频自动生成系统  
- 支持通过输入文本描述词生成相应的视频内容  
- 探索 Text-to-Video 生成、多模态语义理解与视觉内容合成  

### 👁️ Computer Vision

- 图像分类、目标检测与视觉识别任务  
- 基于深度学习的图像特征提取与视觉感知  
- 多模态视觉语言模型相关应用探索  

### 🕹️ AI Agent

- 面向实际任务的智能体系统设计  
- LLM + Tools 的自动化工作流构建  
- 多步骤推理、任务规划与信息检索应用
